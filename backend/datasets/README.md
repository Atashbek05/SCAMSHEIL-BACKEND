# Datasets

| Folder      | Purpose                                                              |
|-------------|----------------------------------------------------------------------|
| `raw/`      | Original, unmodified source files (CSV, JSON). Never edit these.    |
| `processed/`| Cleaned and feature-engineered files ready for model training.       |

## Recommended public datasets to start with

- **SMS Spam Collection** — UCI ML Repository (ham/spam SMS messages)
- **Enron Email Dataset** — phishing/spam emails
- **PhishTank** — verified phishing URLs (API available)

Add any downloaded files to `raw/` and run the preprocessing script in `scripts/`.
